# Dummy

`Dummy` layer does nothing. Literally.

Dummy layer can be useful when you have many destinations of other layers you want to merge into one.

### Settings

<details>
  <summary>JSON view</summary>

```json
{
    "action": "dummy",
    "src": [
        "$data_1"
    ],
    "dst": "$dummy_2",
    "settings": {}
}
```

</details>
